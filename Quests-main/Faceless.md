### STEP 5A: Create Video with FREE TOOLS (CapCut Method)

**What You Need:**
- CapCut (Free - download on phone or PC)
- Your voiceover MP3
- Stock footage from Pexels/Pixabay

**Production Process:**

**1. Gather Visual Assets (15 minutes)**

Go to:
- **Pexels.com** - Free stock videos
- **Pixabay.com** - Free videos and images
- **Coverr.co** - Free B-roll footage

Search terms based on your script:
- If script mentions "money" → search "money, cash, wealth"
- If script mentions "success" → search "business, entrepreneur, office"
- Download 20-30 clips (more than you need)

**2. CapCut Desktop/Mobile Editing:**

**Import:**
- Open CapCut
- Import your voiceover
- Import all video clips

**Timeline:**
- Drag voiceover to timeline
- Listen to first sentence
- Add matching video clip above voiceover
- Cut clip when sentence ends
- Repeat for entire video

**Quick Cuts:**
- Change video every 3-5 seconds
- Keeps viewers engaged
- Use "Split" tool constantly

**Add Text:**
- Use "Text" → "Default"
- Add key phrases on screen
- Yellow text with black outline (most readable)
- Position: Lower third

**Transitions:**
- Select clip → "Transition"
- Use: Dissolve, Slide, Zoom
- Duration: 0.3-0.5 seconds

**Background Music:**
- Go to "Audio" → "Music"
- Choose no-copyright music
- Lower volume: 15-20%
- Don't overpower voiceover

**Export Settings:**
- Resolution: 1080p (minimum)
- Frame Rate: 30fps
- Format: MP4

### STEP 6: Create Killer Thumbnail (10 minutes)

**Go to Canva.com (Free)**

**Size:** 1280 x 720 pixels (YouTube standard)

**Thumbnail Formula:**
```
Bold Text + Emotion Face/Object + Bright Colors + Curiosity Gap
```

**ChatGPT Prompt for Thumbnail Ideas:**

```
"Generate 5 YouTube thumbnail concepts for my video titled: [YOUR TITLE]

For each concept, describe:
- Main visual element
- Text overlay (3-5 words max)
- Color scheme
- Emotional trigger
- What makes it clickable

Style: High contrast, attention-grabbing, mobile-friendly"
```

**Canva Tutorial:**
1. Open Canva → "YouTube Thumbnail"
2. Use template OR create custom
3. Add background image from Pexels
4. Add text:
   - Font: Anton, Bebas Neue, or Impact
   - Size: LARGE (readable on phone)
   - Colors: Yellow, Red, or White
   - Stroke: Black outline (always!)
5. Add arrow/circle to highlight key element
6. Export as PNG

**Pro Thumbnail Tips:**
- Show emotion (shocked face, money, success symbols)
- Use contrast (dark background + bright text)
- Keep it simple (3-5 words max)
- Test on your phone (does it look clear when small?)
