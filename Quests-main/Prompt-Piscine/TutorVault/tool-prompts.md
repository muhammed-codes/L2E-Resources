## Exercise 1: Structured JSON Output

### What to Do
Create prompts that produce machine-readable JSON output.

### Weak → Strong

**Weak (Unstructured):**
> "List three cities with their populations."

**Output:**
> Tokyo has 37 million. Delhi has 30 million. Shanghai has 24 million.

---

**Strong (JSON):**
> "List three cities with their populations. Return in JSON: [{city: string, population: number}]"

**Output:**
```json
[
  {"city": "Tokyo", "population": 37000000},
  {"city": "Delhi", "population": 30000000},
  {"city": "Shanghai", "population": 24000000}
]
```

### Key Phrase
"Return in JSON format" + schema = structured output!

## Exercise 3: Chaining Prompts for Agent Flow

### What to Do
Combine multiple prompts to simulate a tool-using agent.

### The Flow

```
Step 1: User Query → Step 2: API Call → Step 3: Format Answer
```

### Example: Weather Bot

**Step 1 - User Query:**
> "What's the weather in Paris?"

**Step 2 - Simulate API:**
> "Simulate weather API: {location: Paris, temp: 18, condition: Cloudy}"

**Step 3 - Format:**
> "Format this for user: Paris is Cloudy at 18°C"

**Final Output:**
> "The weather in Paris today is cloudy with a temperature of 18°C."

## 📊 Key Concepts

### JSON Schema
Tell AI exactly what fields you want:
```json
{
  "field_name": "type",
  "age": "number",
  "is_active": "boolean"
}
```

### Prompt Chaining
Breaking complex tasks into steps:
1. Input → 2. Process → 3. Output
